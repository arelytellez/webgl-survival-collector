# 🎮 Entorno 3D Interactivo con Three.js

Este proyecto presenta una aplicación web que permite explorar un **entorno tridimensional interactivo** utilizando la librería **Three.js**.

El usuario puede desplazarse dentro del escenario mediante controles **FPS (First Person Shooter)** y experimentar un sistema de **detección de colisiones optimizado mediante Octree**.

La aplicación está desarrollada como una demostración de interacción en ambientes virtuales utilizando **JavaScript, Three.js y Bootstrap 5**.

---

## 🚀 Características

* Navegación en **primera persona (FPS)**
* Control mediante **teclado y mouse**
* Simulación de **física básica**
* **Detección de colisiones con Octree**
* Interfaz web moderna con **Bootstrap 5**
* Página principal con introducción y acceso al demo

---

## 🖥️ Tecnologías utilizadas

* **Three.js** – Renderizado de gráficos 3D en el navegador
* **JavaScript** – Lógica de interacción
* **HTML5** – Estructura de la aplicación
* **CSS3** – Estilos visuales
* **Bootstrap 5** – Diseño responsive
* **UIverse** – Componentes visuales
